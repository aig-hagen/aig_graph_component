## Usage

### Development

##### Install required dependencies:
`npm install`

##### Starting the development server:
`npm run dev`

*For more commands refer to the scripts section in [package.json](./package.json).*

### Web Component
The graph component is also available as a **[custom element](./web-component/README.md)**
which allows you to embed the graph component using the HTML tag `<graph-component/>` as seen in the [index.html](./web-component/index.html) file.

*More details can be found in the [web-component](web-component) directory.*
