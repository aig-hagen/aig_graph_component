# Graph-Component

## Usage
### Web Component / Custom Element
The graph component is available as a **custom element**
which allows you to embed the graph component into an html-file simply by using the tag `<graph-editor/>` as seen in the [index.ce.html](application-example-ce/index.ce.html) file in the *application-example-ce* directory.

## Development
### Project Setup

#### Install required Dependencies
```sh
npm install
```

#### Compile and Hot-Reload for Development

```sh
npm run dev
```

#### Type-Check and Compile for Production

```sh
npm run build
```


For more commands refer to the scripts section in [package.json](./package.json).
